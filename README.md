### ฟังก์ชันการใช้งาน
#### 1. ATtiny 
* เมื่อมีการชน Pin 3 จะ High 2 นาทีเพื่อบอก ESP ว่ามีการชน
* เมื่อ Pin 2 (มี Internal_Pullup) HIGH >>> LOW จะเริ่มทำการกระพริบไฟสีแดงไป 2 นาที

Hardware
============

* KidBright32
* Motor Drive L298N
* Baterry AA 1.2V x 8
